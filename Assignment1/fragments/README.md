2. Navigate to the project directory:

3. Install the dependencies:

### Scripts

The following npm scripts are available:

Start the server:

This will start the server using node src/server.js.

Run in development mode:

This will start the server in development mode with nodemon and set the LOG_LEVEL to debug. The server will automatically restart when files change.

Run in debug mode:

This will start the server in debug mode with nodemon, set the LOG_LEVEL to debug, and enable the inspector on 0.0.0.0:9229.

Lint the code:

This will run eslint on all JavaScript files in the src directory.
